## 💡 使用技巧

### 技巧 1: 快速验证配置

```bash
# 一键测试所有功能
python test_deepseek.py
```

### 技巧 2: 查看日志

```bash
# 管道日志
cat pipeline.log

# 或使用 PowerShell
Get-Content pipeline.log -Tail 50
```

### 技巧 3: 清理数据重新开始

```bash
# 删除临时数据
Remove-Item data/*.json
Remove-Item output/*.html
```

### 技巧 4: 切换 LLM 提供商

编辑 `config.json`:
```json
"llm_provider": "deepseek"  // 改为 "openai" 或 "anthropic"
```
